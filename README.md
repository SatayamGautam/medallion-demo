# Medallion Demo (bronze → silver → gold)

Run this project in Binder: https://mybinder.org/v2/gh/SatayamGautam/medallion-demo/HEAD

1. Open notebooks/01_bronze.ipynb and run top→bottom
2. Run notebooks/02_silver.ipynb then notebooks/03_gold.ipynb
3. Explore data/bronze, data/silver, data/gold

This demo uses pandas + parquet + sqlite to illustrate the medallion architecture without local installs.
